# Music_DJ
>Music_DJ is an easy to use, simple music bot. Play, loop, pause, resume, skip, cloudify, and bass boost songs.  Get started by inviting it to your discord.

Supported music sources:
- YouTube
- Soundcloud
- Mixer
- Twitch
- Bam camp
- And more!

## Commands

   `BOT COMMANDS` | `DESCRIPTION`
----------------- | -----------------
`+about`       => | Info about the bot.
`+bassboost`   => | Bass boosts the current playing song.
`+help`        => | Sends you a list of commands.
`+invite`      => | Sends you the invite link for the bot.
`+join`        => | Joins the voice channel you are in.
`+loop/unloop` => | Loops or unloops the current playing song.
`+reset`       => | Resets the volume back to normal.
`+pause`       => | Pauses the current playing song.
`+ping`        => | Sends you the client's ping and API latency.
`+play`        => | Plays the song you search for in the voice channel you are in.
`+playing`     => | Shows the current playing song.
`+queue`       => | Lists all the songs in the current queue.
`+resume`      => | Resumes the current playing song.
`+skip`        => | Skips the current playing song.
`+stats`       => | Shows stats on the bot.
`+stop`        => | Resumes the current playing song.
`+support`     => | Sends the support server for the bot.
`+earrape`     => | Makes the music very loud and bad quality.
`+volume`      => | Shows/sets the volume of the song.
